## Variable Rules

```python
from flask import Flask
app = Flask(__name__)

@app.route('/')
def index():
    return 'Index Page'

@app.route('/about')
def about():
    return 'About Page'

@app.route('/hello/<name>')
def greeting(name):
	# give the user a personalized greeting
    return '<h1>Hello, %s!</h1>' % name
```
